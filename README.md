
# Distill AI: Умный помощник для заметок

Telegram-бот с поддержкой RAG (Retrieval-Augmented Generation), который индексирует ваши заметки и позволяет получать точные ответы на основе вашего контекста.

---

## 🧠 Ключевые возможности

* **Умный поиск:** Семантический поиск по документам (PDF, TXT, DOCX и др.) через векторную БД ChromaDB.
* **Админ-панель:** Встроенный GUI на **CustomTkinter** для управления ботом:
    * Мониторинг активных пользователей.
    * Просмотр количества заметок пользователей.
    * Очистка данных пользователей из базы.
    * Настройка прокси-сервера прямо из интерфейса.
* **Работа с контекстом:** Автоматическое сжатие и извлечение сути из ваших заметок с помощью **кастомного компрессора**, который сокращает длину промпта без потери смысла.
* **Гибкость:** Поддержка LLM и локальное хранение данных (полная приватность).

---

## 🛠 Технологический стек

* **Язык:** Python 3.10+
* **GUI:** [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter)
* **Векторная БД:** ChromaDB
* **Telegram:** aiogram
* **ML:** Sentence Transformers

## 🚀 Установка и запуск

### 1. Клонируйте репозиторий

```bash
git clone https://github.com/yourusername/distill-ai-notes.git
cd distill-ai-notes
```

### 2. Создайте виртуальное окружение и установите зависимости

```bash
python -m venv venv

# Windows:
venv\Scripts\activate

# Linux/macOS:
source venv/bin/activate

pip install -r requirements.txt
```

### 3. Настройка окружения

Создайте файл `.env` в корне проекта и добавьте:

```env
TELEGRAM_TOKEN=ваш_токен_бота
MISTRAL_API_KEY=ваш_ключ
```

### 4. Запуск

```bash
python main.py
```
